বৃত্ত (Circle)

WB ১২শ শ্রেণী - সম্পূর্ণ সমাধান

সমস্যা ১: তিনটি বিন্দুগামী বৃত্ত
দেওয়া:
বৃত্ত (1, -6), (2, 1) ও (5, 2) বিন্দুগামী।
সমাধান:
বৃত্তের সাধারণ সমীকরণ: \(x^2 + y^2 + 2gx + 2fy + c = 0\)
তিনটি বিন্দু বৃত্তে থাকায়:
• বিন্দু (1, -6): \(1 + 36 + 2g - 12f + c = 0\) ⟹ \(2g - 12f + c = -37\)
• বিন্দু (2, 1): \(4 + 1 + 4g + 2f + c = 0\) ⟹ \(4g + 2f + c = -5\)
• বিন্দু (5, 2): \(25 + 4 + 10g + 4f + c = 0\) ⟹ \(10g + 4f + c = -29\)
সমীকরণ সমাধান করলে: \(g = -5, f = 3, c = 9\)
উত্তর:
  • বৃত্তের সমীকরণ: \(x^2 + y^2 - 10x + 6y + 9 = 0\)
  • কেন্দ্র: (5, -3)
  • ব্যাসার্ধ: \(\sqrt{25 + 9 - 9} = 5\) একক
সমস্যা ২: প্যারামেট্রিক থেকে কার্তেসীয় সমীকরণ
দেওয়া:
\(x = \frac{1}{2}(1 + 5\cos t), \quad y = \frac{1}{2}(-2 + 5\sin t)\)
সমাধান:
\(2x = 1 + 5\cos t\) ⟹ \(\cos t = \frac{2x-1}{5}\)
\(2y = -2 + 5\sin t\) ⟹ \(\sin t = \frac{2y+2}{5}\)

\(\cos^2 t + \sin^2 t = 1\) ব্যবহার করে:
\[\left(\frac{2x-1}{5}\right)^2 + \left(\frac{2y+2}{5}\right)^2 = 1\]
উত্তর:
  • \((2x-1)^2 + (2y+2)^2 = 25\)
  • বা: \(x^2 + y^2 - x + 2y - 5 = 0\)
  • কেন্দ্র: \(\left(\frac{1}{2}, -1\right)\)
  • ব্যাসার্ধ: \(\frac{5}{2}\) একক
সমস্যা ৩: ত্রিভুজ ও বৃত্ত সংক্রান্ত প্রমাণ
দেওয়া:
দুটি বাহু অক্ষে অবস্থিত, তৃতীয় বাহু \(x^2 + y^2 - 2ax - 2ay + a^2 = 0\) বৃত্তকে স্পর্শ করে।
প্রমাণ:
বৃত্তটি: \((x-a)^2 + (y-a)^2 = a^2\), কেন্দ্র (a, a), ব্যাসার্ধ a।

ত্রিভুজের বাহু দুটি x ও y অক্ষে থাকায়, এক শীর্ষ মূলবিন্দুতে, অন্য দুটি (p, 0) ও (0, q) এ।

তৃতীয় বাহু (পরিধি): \(\frac{x}{p} + \frac{y}{q} = 1\) ⟹ \(qx + py - pq = 0\)

এই রেখা বৃত্তকে স্পর্শ করায়, মূলবিন্দু থেকে দূরত্ব = a:
\[\frac{|pq|}{\sqrt{p^2 + q^2}} = a\] পরিকেন্দ্র হল পরিধির মধ্যবিন্দু: \(\left(\frac{p}{2}, \frac{q}{2}\right)\)

\(x = \frac{p}{2}, y = \frac{q}{2}\) হলে \(p = 2x, q = 2y\)

শর্ত থেকে: \(\frac{4xy}{\sqrt{4x^2 + 4y^2}} = a\) ⟹ \(\frac{xy}{\sqrt{x^2 + y^2}} = \frac{a}{2}\)
অতএব, সঞ্চারপথ:
\[\boxed{a^2 - 2a(x + y) + 2xy = 0}\]
সমস্যা ৪: একটি বৃত্ত অন্যটির মধ্যে অবস্থিত
দেওয়া:
\(x^2 + y^2 - 2y - 15 = 0\) এবং \(x^2 + y^2 - x - 30 = 0\)
প্রমাণ:
প্রথম বৃত্ত: \(x^2 + (y-1)^2 = 16\), কেন্দ্র C₁(0, 1), ব্যাসার্ধ r₁ = \(\sqrt{9 + 1 - 8} = 4\)

দ্বিতীয় বৃত্ত: \(\left(x - \frac{1}{2}\right)^2 + y^2 = \frac{121}{4}\), কেন্দ্র C₂\(\left(\frac{1}{2}, 0\right)\), ব্যাসার্ধ r₂ = \(\frac{11}{2}\)

দুই কেন্দ্রের দূরত্ব: \(d = \sqrt{\frac{1}{4} + 1} = \frac{\sqrt{5}}{2} \approx 1.12\)

ছেদের শর্ত: \(|r_1 - r_2| < d < r_1 + r_2\)
যেহেতু \(d + r_1 = \frac{\sqrt{5}}{2} + 4 < \frac{11}{2} = r_2\) এবং \(d < r_2 - r_1\)
অতএব:
প্রথম বৃত্ত সম্পূর্ণভাবে দ্বিতীয়টির মধ্যে অবস্থিত। ✓
সমস্যা ৫: মূলবিন্দুগামী বৃত্ত
দেওয়া:
মূলবিন্দু দিয়ে যায়, y = x ও y = -x থেকে a দৈর্ঘ্যের জ্যা ছিন্ন করে।
সমাধান:
মূলবিন্দুগামী বৃত্ত: \(x^2 + y^2 + 2gx + 2fy = 0\)

রেখা y = x থেকে দূরত্ব: \(\frac{|2g - 2f|}{\sqrt{2}} = d_1\)

জ্যার দৈর্ঘ্য = \(2\sqrt{R^2 - d_1^2} = a\)

যেখানে \(R = \sqrt{g^2 + f^2}\)
উত্তর:
  • \(x^2 + y^2 + \frac{a}{\sqrt{2}}x + \frac{a}{\sqrt{2}}y = 0\)
  • অথবা: \(x^2 + y^2 - \frac{a}{\sqrt{2}}x - \frac{a}{\sqrt{2}}y = 0\)
সমস্যা ৬: অর্ধবৃত্তস্থ কোণ - জ্যামিতিক প্রমাণ
প্রমাণ:
ধরুন বৃত্তের কেন্দ্র O, ব্যাস AB, পরিধিস্থ বিন্দু C। স্থানাঙ্ক: O(0,0), A(-r,0), B(r,0), C(r cos θ, r sin θ)

ভেক্টর: \(\vec{CA} = (-r - r\cos θ, -r\sin θ)\), \(\vec{CB} = (r - r\cos θ, -r\sin θ)\)

\(\vec{CA} \cdot \vec{CB} = (-r - r\cos θ)(r - r\cos θ) + (-r\sin θ)(-r\sin θ)\)

\(= -r^2(1 + \cos θ)(1 - \cos θ) + r^2\sin^2 θ\)

\(= -r^2(1 - \cos^2 θ) + r^2\sin^2 θ = -r^2\sin^2 θ + r^2\sin^2 θ = 0\)
অতএব:
∠ACB = 90° ✓
সমস্যা ৭: সরলরেখার ছেদ বিন্দুর সঞ্চারপথ
দেওয়া:
\(x\cos t + y\sin t = a\) এবং \(x\sin t - y\cos t = b\)
সমাধান:
প্রথম সমীকরণ × cos t: \(x\cos^2 t + y\sin t \cos t = a\cos t\) ... (1)

দ্বিতীয় সমীকরণ × sin t: \(x\sin^2 t - y\cos t \sin t = b\sin t\) ... (2)

(1) + (2): \(x(\cos^2 t + \sin^2 t) = a\cos t + b\sin t\)

⟹ \(x = a\cos t + b\sin t\) ... (3)

একই পদ্ধতিতে: \(y = a\sin t - b\cos t\) ... (4)
(3) এবং (4) থেকে:
\[x^2 + y^2 = (a\cos t + b\sin t)^2 + (a\sin t - b\cos t)^2\] \[= a^2\cos^2 t + b^2\sin^2 t + a^2\sin^2 t + b^2\cos^2 t = a^2 + b^2\]
সঞ্চারপথ:
\[\boxed{x^2 + y^2 = a^2 + b^2}\]
(একটি বৃত্ত)
সমস্যা ৮: x-অক্ষ স্পর্শকারী বৃত্ত
দেওয়া:
x-অক্ষকে (2, 0) এ স্পর্শ করে, y-অক্ষ থেকে 4 একক জ্যা ছিন্ন করে।
সমাধান:
x-অক্ষ স্পর্শ করলে কেন্দ্র (2, r), ব্যাসার্ধ r।

সমীকরণ: \((x-2)^2 + (y-r)^2 = r^2\)

y-অক্ষে (x = 0): \(4 + (y-r)^2 = r^2\)

⟹ \((y-r)^2 = r^2 - 4\)

জ্যার দৈর্ঘ্য = \(2\sqrt{r^2 - 4} = 4\)

⟹ \(r^2 - 4 = 4\) ⟹ \(r = 2\sqrt{2}\)
উত্তর:
  • \((x-2)^2 + (y - 2\sqrt{2})^2 = 8\)
  • অথবা: \((x-2)^2 + (y + 2\sqrt{2})^2 = 8\)
  • প্রসারিত: \(x^2 + y^2 - 4x \mp 4\sqrt{2}y + 4 = 0\)
সমস্যা ৯: অক্ষ স্পর্শকারী বৃত্ত - প্রমাণ
দেওয়া:
বৃত্ত উভয় অক্ষকে স্পর্শ করে এবং কেন্দ্র lx + my + n = 0 সরলরেখায় থাকে।
প্রমাণ:
বৃত্ত উভয় অক্ষকে স্পর্শ করলে কেন্দ্র (a, a) এবং ব্যাসার্ধ = a।

সমীকরণ: \((x-a)^2 + (y-a)^2 = a^2\)

সম্প্রসারণ: \(x^2 + y^2 - 2ax - 2ay + a^2 = 0\)

কেন্দ্র (a, a) রেখা lx + my + n = 0 এ থাকায়:
\(la + ma + n = 0\) ⟹ \(a(l+m) = -n\) ⟹ \(a = \frac{-n}{l+m}\)

সমীকরণে a প্রতিস্থাপন করে গুণ করুন \((l+m)^2\):
অতএব:
\[\boxed{(l+m)^2(x^2+y^2) + 2n(l+m)(x+y) + n^2 = 0}\]
সমস্যা ১০: দুটি বৃত্তের স্পর্শ শর্ত
দেওয়া:
\(x^2 + y^2 + 2gx + 2fy = 0\) এবং \(x^2 + y^2 + 2g'x + 2f'y = 0\)
প্রমাণ:
প্রথম বৃত্ত: কেন্দ্র C₁(-g, -f), ব্যাসার্ধ r₁ = \(\sqrt{g^2 + f^2}\)

দ্বিতীয় বৃত্ত: কেন্দ্র C₂(-g', -f'), ব্যাসার্ধ r₂ = \(\sqrt{g'^2 + f'^2}\)

স্পর্শ করলে: \(d = |r_1 \pm r_2|\)

\(\sqrt{(g-g')^2 + (f-f')^2} = \sqrt{g^2 + f^2} \pm \sqrt{g'^2 + f'^2}\)

বর্গ করে সরল করলে: \((gg' + ff')^2 = (g^2+f^2)(g'^2+f'^2)\)
অতএব:
\[\boxed{gf' = g'f}\]
সমস্যা ११: দুটি বৃত্তের ছেদ যাচাই
দেওয়া:
\(x^2 + y^2 + 6x + 2y + 8 = 0\) এবং \(x^2 + y^2 + 2x + 6y + 1 = 0\)
সমাধান:
প্রথম বৃত্ত: কেন্দ্র C₁(-3, -1), ব্যাসার্ধ r₁ = \(\sqrt{9 + 1 - 8} = \sqrt{2}\)

দ্বিতীয় বৃত্ত: কেন্দ্র C₂(-1, -3), ব্যাসার্ধ r₂ = \(\sqrt{1 + 9 - 1} = 3\)

কেন্দ্রের দূরত্ব: \(d = \sqrt{4 + 4} = 2\sqrt{2}\)

ছেদের শর্ত: \(|r_1 - r_2| < d < r_1 + r_2\)

\(3 - \sqrt{2} < 2\sqrt{2} < 3 + \sqrt{2}\)

\(1.586 < 2.828 < 4.414\) ✓
অতএব:
বৃত্ত দুটি পরস্পর ছেদ করে।
সমস্যা १२: সমবাহু ত্রিভুজের অন্তর্বৃত্ত
দেওয়া:
অন্তর্বৃত্ত সমীকরণ: \(2x^2 + 2y^2 + 3x - y - 5 = 0\)
সমাধান:
\(x^2 + y^2 + \frac{3}{2}x - \frac{1}{2}y - \frac{5}{2} = 0\)

মান: g = 3/4, f = -1/4, c = -5/2

ব্যাসার্ধ: \(r = \sqrt{\frac{9}{16} + \frac{1}{16} + \frac{5}{2}} = \sqrt{\frac{50}{16}} = \frac{5\sqrt{2}}{4}\)

সমবাহু ত্রিভুজে: অন্তঃব্যাসার্ধ r = \(\frac{a}{2\sqrt{3}}\) যেখানে a = বাহু

a = \(2\sqrt{3}r = \frac{5\sqrt{6}}{2}\)

ক্ষেত্রফল = \(\frac{\sqrt{3}}{4}a^2 = \frac{\sqrt{3}}{4} \cdot \frac{150}{4}\)
উত্তর:
\[\text{ক্ষেত্রফল} = \frac{75\sqrt{3}}{8} \text{ বর্গ একক}\]
সমস্যা १३: সাধারণ জ্যার দৈর্ঘ্য - প্রমাণ
দেওয়া:
\((x-a)^2 + (y-b)^2 = c^2\) এবং \((x-b)^2 + (y-a)^2 = c^2\)
প্রমাণ:
দুটি সমীকরণ বিয়োগ করলে:

\((x-a)^2 - (x-b)^2 + (y-b)^2 - (y-a)^2 = 0\)

সরল করলে: \(x = y\) (সাধারণ জ্যা)

প্রথম বৃত্তে কেন্দ্র (a,b), ব্যাসার্ধ c।

রেখা x = y থেকে দূরত্ব = \(\frac{|a-b|}{\sqrt{2}}\)

জ্যার দৈর্ঘ্য = \(2\sqrt{c^2 - \frac{(a-b)^2}{2}} = \sqrt{4c^2 - 2(a-b)^2}\)
অতএব:
\[\text{দৈর্ঘ্যের বর্গ} = \boxed{4c^2 - 2(a-b)^2}\]
সমস্যা १४: বৃত্তে অন্তর্লিখিত সমবাহু ত্রিভুজ
দেওয়া:
বৃত্ত: \(x^2 + y^2 + 2gx + 2fy + c = 0\)
সমাধান:
ব্যাসার্ধ: \(R = \sqrt{g^2 + f^2 - c}\)

সমবাহু ত্রিভুজ অন্তর্লিখিত হলে: বাহু = \(R\sqrt{3}\)

ক্ষেত্রফল = \(\frac{\sqrt{3}}{4}(R\sqrt{3})^2 = \frac{\sqrt{3}}{4} \cdot 3R^2 = \frac{3\sqrt{3}}{4}R^2\)
উত্তর:
\[\text{ক্ষেত্রফল} = \frac{3\sqrt{3}}{4}(g^2 + f^2 - c) \text{ বর্গ একক}\]
সমস্যা १५: ব্যাসের সমীকরণ থেকে বৃত্ত
দেওয়া:
(-2, 2) বিন্দুগামী, ব্যাস সমীকরণ: 3x + y = 5 এবং x + y + 1 = 0
সমাধান:
দুই ব্যাসের সংযোগ বিন্দুতে কেন্দ্র অবস্থিত:

\(3x + y = 5\) এবং \(x + y = -1\)

বিয়োগ করলে: \(2x = 6\) ⟹ \(x = 3\)

\(y = -1 - 3 = -4\)

কেন্দ্র: (3, -4)

ব্যাসার্ধ = দূরত্ব (-2, 2) থেকে (3, -4):
\(r = \sqrt{25 + 36} = \sqrt{61}\)
উত্তর:
  • বৃত্তের সমীকরণ: \((x-3)^2 + (y+4)^2 = 61\)
  • অথবা: \(x^2 + y^2 - 6x + 8y - 36 = 0\)
  • ব্যাসার্ধ: \(\sqrt{61}\) একক
সমস্যা १६: বৃত্তের পরিবার ও সঞ্চারপথ
দেওয়া:
\(x^2 + y^2 - x(3p+4) - y(p-2) + 10p = 0\)
অংশ 1: (3, 1) বিন্দুগামী প্রমাণ
(3, 1) প্রতিস্থাপন:

\(9 + 1 - 3(3p+4) - 1(p-2) + 10p = 0\)

\(10 - 9p - 12 - p + 2 + 10p = 0\)

\(0 = 0\) ✓ (সর্বদা সত্য)
অংশ 2: কেন্দ্রের সঞ্চারপথ
কেন্দ্র: \(\left(\frac{3p+4}{2}, \frac{p-2}{2}\right)\)

ধরুন h = \(\frac{3p+4}{2}\), k = \(\frac{p-2}{2}\)

⟹ \(p = 2k + 2\)

\(h = \frac{3(2k+2)+4}{2} = \frac{6k+10}{2} = 3k + 5\)
উত্তর:
\[\text{সঞ্চারপথ:} \quad x = 3y + 5 \quad \text{অথবা} \quad x - 3y - 5 = 0\]
সমস্যা १७: তিনটি বৃত্ত - কেন্দ্র সংযোগী ও ব্যাসার্ধ
দেওয়া:
\(x^2 + y^2 = 1\), \(x^2 + y^2 + 6x - 2y - 6 = 0\), \(x^2 + y^2 - 12x + 4y - 9 = 0\)
সমাধান:
প্রথম বৃত্ত: কেন্দ্র (0, 0), ব্যাসার্ধ r₁ = 1

দ্বিতীয় বৃত্ত: কেন্দ্র (-3, 1), ব্যাসার্ধ r₂ = \(\sqrt{9 + 1 + 6} = 4\)

তৃতীয় বৃত্ত: কেন্দ্র (6, -2), ব্যাসার্ধ r₃ = \(\sqrt{36 + 4 + 9} = 7\)
কেন্দ্রগুলি একরেখীয় যাচাই:
(0,0), (-3,1), (6,-2) এর ঢাল:

• (0,0) থেকে (-3,1): \(\frac{1-0}{-3-0} = -\frac{1}{3}\)

• (-3,1) থেকে (6,-2): \(\frac{-2-1}{6-(-3)} = \frac{-3}{9} = -\frac{1}{3}\)

✓ একরেখীয়

✓ সমান্তর প্রগতি
সমস্যা १८: y-অক্ষ স্পর্শকারী বৃত্ত
দেওয়া:
y-অক্ষকে মূলবিন্দুতে স্পর্শ করে, (a, b) বিন্দুগামী।
সমাধান:
y-অক্ষ স্পর্শ করলে কেন্দ্র (r, k) আকারের, ব্যাসার্ধ r।

মূলবিন্দুতে স্পর্শ করলে k = 0, অতএব কেন্দ্র (r, 0)।

সমীকরণ: \((x-r)^2 + y^2 = r^2\)

(a, b) বিন্দুতে: \((a-r)^2 + b^2 = r^2\)

\(a^2 - 2ar + r^2 + b^2 = r^2\)

\(a^2 + b^2 = 2ar\) ⟹ \(r = \frac{a^2 + b^2}{2a}\)
উত্তর:
\[\left(x - \frac{a^2+b^2}{2a}\right)^2 + y^2 = \left(\frac{a^2+b^2}{2a}\right)^2\]
অথবা: \(x^2 + y^2 - \frac{a^2+b^2}{a}x = 0\)
সমস্যা १९: গুণাঙ্ক নির্ধারণ
দেওয়া:
\(2x^2 + 2y^2 + ax + by + c = 0\), কেন্দ্র (3, -4)
সমাধান:
মান দ্বারা ভাগ করুন 2: \(x^2 + y^2 + \frac{a}{2}x + \frac{b}{2}y + \frac{c}{2} = 0\)

কেন্দ্র: \(\left(-\frac{a}{4}, -\frac{b}{4}\right) = (3, -4)\)

⟹ \(-\frac{a}{4} = 3\) ⟹ a = -12

⟹ \(-\frac{b}{4} = -4\) ⟹ b = 16
সমস্যা २०: অক্ষের ছেদাংশ থেকে বৃত্ত
দেওয়া:
3x + 4y = 12 রেখার অক্ষ ছেদাংশ ব্যাস করে বৃত্ত, মূলবিন্দুগামী।
সমাধান:
রেখা 3x + 4y = 12 এ:

• x-অক্ষে (y = 0): (4, 0)

• y-অক্ষে (x = 0): (0, 3)

ব্যাস সমাপ্তিবিন্দু: (4, 0) এবং (0, 3)

ব্যাসের মধ্যবিন্দু (কেন্দ্র): (2, 3/2)

ব্যাসার্ধ: \(r = \frac{1}{2}\sqrt{16 + 9} = \frac{5}{2}\)

মূলবিন্দুতে যাচাই:
\((0-2)^2 + \left(0-\frac{3}{2}\right)^2 = 4 + \frac{9}{4} = \frac{25}{4}\) ✓
